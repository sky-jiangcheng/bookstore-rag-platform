### 1️⃣ ServiceOrchestrator - 统一服务编排

**核心职责**:
- 统一编排所有业务服务调用

**优势**:
- ✅ 减少 API 层代码复杂度
- ✅ 统一的错误处理
- ✅ 自动缓存管理
- ✅ 性能监控

### 2️⃣ CircuitBreaker - 熔断机制

**设计特点**:
- **三态转换**: CLOSED → OPEN → HALF_OPEN → CLOSED
- **自动恢复**: 半开状态尝试恢复
- **故障检测**: 基于失败次数和时间窗口
- **降级支持**: 自动调用 fallback 函数

### 3️⃣ ConversationManager - 多轮对话管理

**核心组件**:
- **Message** - 单条消息
- **ConversationContext** - 对话上下文
- **ConversationManager** - 管理器
- **MultiRoundConversationService** - 服务

### 4️⃣ Celery 异步任务队列
